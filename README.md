## selector  :- 
**color - red**
1. **tag selector (div)  -- page div tag** 
```css
<div>Hello World </div>
syntax:-
<style type="text/css">
div{
color:"red";
}
</style>
```
2. **id selector (unique) - (#idselectorName)**

```css
<div id="helloworld">Hello World </div>
syntax:-
<style type="text/css">
#helloworld{
color:"red";
}
</style>
```

3. **class Selector (.SelectorName)** - bluk
```css
<div class="helloworld">Hello World </div>

syntax:-
<style type="text/css">
.helloworld{
color:"red";
}
</style>
```
